Website is being developed using a mobile approach. The design is responsive enough to be viewable on all screen sizes. Good color contrast and standard font size have been used to increase the readability of the website.

To improve search engine optimization, suitable page title is defined for website and meta tags are used, i.e. keyword and description.

Website is being developed by keeping in mind all the 4 accessibilty principles of WCAG 2.1 i.e. Perceivable, Operable, Understandable and Robust.
